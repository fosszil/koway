# koway
Koway, an open‑source, community‑driven city transport & travel app for Coimbatore built in Flutter.

## Features
- Get info on buses between two locations
- View bus routes by their numbers and their destinations
- Map view of travel routes [upcoming feature]

## Usage
This app is built on Flutter and the flutter files are in the folder koway

- Run `flutter run` to directly run the app for development purposes
- Run `flutter build` to build the app for production and finished feel

## Contributing

We are currently accepting changes for adding routes and editing existing routes. Feel free to add or work on new features/bugs.
